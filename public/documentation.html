Movie Information

Request URL Response
GET /movies Returns a list of all movies
GET /movies/:title Returns data for a single movie by title
GET /genre/:genreName Returns description for a given genre
GET /director/:name Returns data about a movie director by name
POST /users Registers a new user
PUT /users/:username Updates a user's username
POST /users/:username/movies/:movieTitle Adds a movie to user's favorites
DELETE /users/:username/movies/:movieTitle Removes a movie from user's favorites
DELETE /users/:username Deregisters a user
